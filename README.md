ДЗ (Модуль 8)
В папке '8-weather-extended' реализовать CLI-утилиту с точкой входа 'weather.js', которая будет выводить погоду для нескольких городов, используя стороннее API (https://openweathermap.org):
1. Реализовать функциональность ввода латиницей в виде строки 'node weather.js -s [CITY1 CITY2 ...]' названий городов и сохранения в json-файл для дальнейшего использования городов по умолчанию.
2. Реализовать функциональность ввода латиницей в виде строки 'node weather.js -l [LANG]' (например, английский [EN] или русский [RU]) языка вывода информации о погоде и сохранения в json-файл для дальнейшего использования языка по умолчанию.
3. Описать функционал CLI-утилиты при вводе строки 'node weather.js -h'.
